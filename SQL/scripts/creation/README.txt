Здесь собраны SQL-скрипты для создания таблиц в базах данных РУЖЭЛЬ.

Две директории содержат скрипты для соответствующих СУБД.
В них каждый файл *sql соответствует отдельной базе данных.
В директориях RujelStatic и RujelYear собраны скрипты создания таблиц в этих двух базах.

В каждой создаваемой базе данных сначала должны быть созданы системные таблицы,
описанные в скрипте init_tables.sql
