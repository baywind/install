Здесь лежат файлы скриптов SQL

Начиная с версии 0.9.9 РУЖЭЛЬ умеет самостоятельно создавать для себя базы данных в СУБД MySQL и PostgreSQL.
Для начала работы необходимо только создать пользователя СУБД и указать в настройке РУЖЭЛЬ его имя и пароль.
Пользователь должен обладать правом создания БД с нужными именами.
Команды для создания пользователя вы указаны в файлах с именами, соответствующими СУБД.

Для нештатных ситуаций в директории scripts собраны SQL-скрипты, используемые РУЖЭЛЬ 
для создания и обновления баз данных.
