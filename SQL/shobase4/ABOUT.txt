для использования БД "Типовое Ядро" Всеобуч, Кадры:

shoStruct.sql - упрощенноая структура базы данных "Типовое Ядро" (БД-списки)
shoStudent.sql - шаблон для заполнения БД учеников и классов
shoTeacher.sql - шаблон для заполнения БД учителей

при использовании резервной копии Типового Ядра (Всеобуч, Кадры) в начало файла нужно вставить следующий текст:

SET NAMES cp1251;
CREATE DATABASE IF NOT EXISTS `shobase4` DEFAULT CHARACTER SET utf8;
USE `shobase4`;
