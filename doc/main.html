Общие настройки

Файл rulel.plist

  • schoolName - Название ОУ для отображения на странице ввода пароля
  • schoolNumber - числовой идентификатор ОУ.
    Используется для размещения нескольких школ на одном сервере. Этот номер должен совпадать с номером SCHOOL_NUM в базе данных в таблицах RujelStatic.PL_CYCLE и RujelYear%s.CU_REASON
  • loggingProperties - указывает на расположение файла настройки лога. Если не указан, используются системные настройки лога Java. Данный файл форматируется в соответствии со спецификацией Java logging
  • diaryLogging - аналогично указывает на настройки лога дневника
  • edu - Настройки параметров учебного процесса
    • Большинство настроек из этого раздела будут перенесены в таблицу базы данных RujelStatic.BASE_SETTINGS
    • minGrade, maxGrade - минимальный и максимальный номера потока, которые должны отображаться в учебном плане
    • newYearMonth, newYearDay - месяца и дата (в числовом виде), которые определяют границу учебного года (по умолчанию, 1 июля)
