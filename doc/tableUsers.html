Настройка пользователей в таблице БД

Файл TableUsers.plist

  • moduleClass - net.rujel.user.UserModule
  • sort - 50
  • dbConnection -

      • актуальный раздел: auth

      • loginHandlerClass - имя класса java, который используется для аутентикации пользователей: net.rujel.user.TableLoginHandler
      • parentLoginHandler - если таблица используется как прокладка, имя класса java, который используется для аутентикации пользователей в родительской системе.
        Cюда копируется значение loginHandlerClass из другого описания пользователей. Если не заполнено, пароли сохраняются в той же таблице
      • passwordDigestAlgorithm - алгоритм шифрования пароля при сохранении в БД (по умолчанию - MD5)
      • autoAddUsers - добавлять в таблицу пользоватетей из родительской системы (по умолчанию - false)
      • readFromParent - использовать информацию о пользователе из родительской системы (по умолчанию - false)
