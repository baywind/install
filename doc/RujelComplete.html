RujelComplete

Модуль создания свертки в конце учебного года

Файл RujelComplete.plist

Описание в процессе доработки.

  • moduleClass
  • sort
  • edu - настройка сохранения свертки
    • studentsCompleteDir - путь до папки, в которую должны складываться файлы по учащимся при свертке учебного года
    • coursesCompleteDir - путь до папки, в которую должны складываться файлы по учебным курсам при свертке учебного года
    • completeDir - путь до папки, в которую должны складываться все файлы при свертке учебного года (в случае если не указаны предыдущие два параметра)
