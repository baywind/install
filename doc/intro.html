Введение

Параметры настройки приложений РУЖЭЛЬ обычно размещаются в директории LOCALROOT/Library/WebObjects/Configuration/rujel

Они находятся в файле rujel.plist и в папке modules, хранящей настройки различных модулей РУЖЭЛЬ. По структуре разделов все файлы настроек идентичны, но различаются по составу параметров.

Логика настроек РУЖЭЛЬ опирается на множество параметров, предусмотренных в поставке по умолчанию. Это значит, что для его полноценной работы достаточно внести минимальные изменения в данные конфигурации — только те, которые невозможно предусмотреть заранее. Если в процессе эксплуатации возникнет желание изменить какие-то параметры, действующие по умолчанию, их нужно внести в один из файлов конфигурации — эти изменения перекроют значения по умолчанию и будут действовать, пока Вы не измените или не удалите свои добавления.

Все файлы поставляются с минимально необходимыми настройками, которые нужно указать. Для изменения дополнительных параметров необходимо самостоятельно добавить соответствующие ключи в соответствии с описанием.

Логика разбиения параметров настройки на различные файлы предусмотрена для упрощения ориентации в них. В файле rujel.plist хранятся общие настройки, а в отдельные файлы вынесены

  • настройки каждого модуля РУЖЭЛЬ — в отдельный одноименный файл настроек,
  • варианты настроек подключения к базам данных,
  • варианты авторизации пользователей РУЖЭЛЬ

Причем, для различных вариантов представлены отдельные заготовки — это упростит выбор нужного Вам варианта.

Файлы-заготовки находятся в папке modules_distr и сгруппированы в папки по назначению:

  • required — разделы настройки, необходимые для корректной работы
  • recommended — готовые к использованию файлы, предлагаемые к подключению изначально
  • database — варианты подключения к базе данных: MySQL/ Firebird ...
  • users — список пользователей из локального файла или разные варианты LDAP
  • zavuch — модули РУЖЭЛЬ-завуч
  • extra — дополнительные модули

Чтобы быть уверенными в соответствии файлов настройки последней версии РУЖЭЛЬ, рекомендуем брать их из последнего дистрибутива.

Для подключения и отключения модулей достаточно поместить в папку modules (или удалить из нее) соответствующий файл. Одни файлы могут быть использованы без изменений, другие требуют предварительной доработки — указания, например, IP-адреса серверов, имен и паролей для доступа к базе данных, параметров подключения к LDAP или отправки электронной почты. Условия использования файлов и правила указания настроек даны в описании.

Модули, требующие предварительной настройки отмечены красной звездочкой *. Требующие настройки параметры выделены красным цветом.
