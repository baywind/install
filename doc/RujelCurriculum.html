Файл RujelCurriculum.plist

Описание в процессе доработки.

  • moduleClass
  • sort
  • dbConnection -
    • edu - настройка автовыставления
      • disablePlanFactCheck - отключить автоматическую проверку заполнения журнала (по умолчанию false)
      • planFactLagDays - задержка (в сутках) автоматической проверки заполнения журнала относительно нулевого дня недели
      • planFactCheckTime - время автоматической проверки заполнения журнала. по умолчанию 0:59
      • joinFactor - коэффициент умножения ставки при совмещении урока (по умолчанию, 0.5)
