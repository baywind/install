Русский язык Добавить 3 3 101 102
Курс еще не создан, в таблице показано расчетное количество часов за учебный цикл.
Цикл обычно равен неделе, но может быть иным, например, 2-недельным.
Литература Антонова А. А. 101 102 Курс создан и равномерно распределен на весь учебный год: в правом поле каждой ячейки показано расчетное количество часов за цикл. Детализации для данного курса не указано (левые поля пусты).
Немецкий язык Пенчук С. Т. 101 80 Курс создан, но не может быть равномерно распределен на весь год, т.к. плановое количество часов не делится нацело на количество учебных циклов. В этой ситуации необходимо указать детализацию нагрузки по отчетным периодам.
Английский язык Апухтина Е. Ф. 136 136
Курс создан и для него корректно заполнена детализация нагрузки по отчетным периодам:
- в левом поле - часы за весь период,
- в правом - часы за один цикл.
Савченко Э. В. 119 Цвет указывает на ошибку в детализации: суммарное количество часов получается меньше планового
Скрипка Е. С. 150 Цвет указывает на ошибку в детализации: суммарное количество часов получается больше планового
Французский язык Торская Е. Р. 102 102
В первом семестре расчетное количество часов превышает плановое.
Во втором - расчетное количество часов меньше планового
